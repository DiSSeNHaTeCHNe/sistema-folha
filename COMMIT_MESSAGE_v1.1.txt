Implementada nova funcionalidade de visualização do organograma em modo
mapa mental interativo usando ReactFlow.

## Novos Recursos

- Toggle para alternar entre visualização Lista (tradicional) e Gráfico (novo)
- Navegação intuitiva com zoom/pan usando mouse
- MiniMap para overview e navegação rápida pela estrutura completa
- Layout automático hierárquico com centralização inteligente de nós
- Controles de zoom (+/-), fit view e lock
- Nós customizados exibindo todas as informações
- Todas funcionalidades de edição mantidas no modo gráfico

## Implementação

- Criado componente `OrganogramaGrafico` com ReactFlow
- Integrado toggle no componente principal `Organograma`
- Algoritmo de layout hierárquico automático
- Conversão de estrutura de árvore para nós/edges do ReactFlow
- Performance otimizada para hierarquias complexas

## Dependências

- Adicionada: reactflow@^11.x
